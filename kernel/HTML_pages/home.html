              Welcome To The Plasma




    This web server is running on Plasma CPU which was made by Steve Rhoads. Also it is working under Plasma Real Time Operating system (RTOS).The Plasma CPU is written in VHDL and implemented on Xilinx Spartan-3E. More documentation about this CPU could be found here: Plasma.

    The Plasma CPU was created from scratch using VHDL (a hardware design language). It is compatible with the MIPS I(TM) instruction set. The Plasma CPU has 32 32-bit registers and is currently running at 25 MHz on a Xilinx FPGA.Additional libraries include an ANSI-C library, a single precision floating point library, and a serial port driver. The ANSI-C library and serial port driver generate an additional 7 KB of executable code.

The Plasma Real Time Operation System (RTOS) was created to run on the Plasma CPU. It supports interrupts, threads, semaphores, message queues, timers, heaps, and pre-emptive context switching. The RTOS is written in about 1500 lines of C and 240 lines of assembly that compiles to less than 8 KB of code. Additional libraries include an ANSI-C library, a single precision floating point library, and a serial port driver. The ANSI-C library and serial port driver generate an additional 7 KB of executable code.

This web server executes the Plasma TCP/IP protocol stack. The Plasma TCP/IP protocol stack supports Ethernet, DHCP, ARP, PING, IP, UDP, TCP, and DNS. The Spartan-3E board uses the Ethernet PHY. The Spartan-3 board instead uses the UART and the PC WinPcap library.
